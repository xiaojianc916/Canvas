# Docs

## Planned structure

```
docs/
├── architecture/          # Overview, dependency rules, state ownership, native boundaries
├── domain-map/            # Context maps and ownership matrices per bounded context
├── adr/                   # Architecture Decision Records
├── rfcs/                  # Request For Comments (pre-ADR proposals)
├── runbooks/              # Operational procedures (debugging, recovery, debugging)
└── compatibility/         # Format version compatibility matrix
```

## Activation phase

Phase 0 (architecture, domain-map) → ongoing (adr, rfc) → Phase 3+ (runbooks, compatibility)
